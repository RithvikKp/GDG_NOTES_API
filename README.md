# Notes Management API

A backend API for managing personal notes, built with Flask and SQLite. It features JWT-based authentication, password hashing, and Role-Based Access Control (RBAC).

## Setup Instructions

1. **Ensure Python 3 is installed** on your system.
2. **Clone/Download** this repository to your local machine.
3. **Install the required dependencies** using the provided requirements file:
   ```bash
   pip install -r requirements.txt
